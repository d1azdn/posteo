User related endpoint

GET api/v1/userall [ADMIN]

Check all user, admin role needed.

Expected data sent :

Web params

Expected data receive :

{ status : "success", message : "All username has been found.", data : users }

GET api/v1/user/:username

Check user using username.

Expected data sent :

Web params

Expected data receive :

{ status : "success", message : "Username has been found.", data : user }

POST api/v1/register

Register using username and password.

Expected data sent :

{ username : "USERNAME", password : "PASSWORD" }

Expected data receive :

{ status : "success", message : "Account has been created." }

POST api/v1/login

Login using username and password.

Expected data sent :

{ username : "USERNAME", password : "PASSWORD" }

Expected data receive :

{ status : "success", message : "Have fun login!", token : "Bearer JWT-TOKEN" }

Post related endpoint

Expected data sent :

{ title : "TITLE", content : "CONTENT" }

Expected data receive :
